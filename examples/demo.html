Data transformation and interactive chart generation.

üîÑ Demo 3: Complete Pipeline with Joins

Complex pipeline with data joins and enrichment.

etl({ enableCache: true, enableParallel: true, enableErrorRecovery: true }) .extract.api('https://jsonplaceholder.typicode.com/posts') .join.api('https://jsonplaceholder.typicode.com/users', { key: 'userId', mode: 'nested' }) .enrich(async (post) => { const response = await fetch(`https://jsonplaceholder.typicode.com/comments?postId=${post.id}`); const comments = await response.json(); return { ...post, commentCount: comments.length }; }) .filter(post => post.commentCount > 3) .load.chart('pie', { x: 'userId', y: 'commentCount', title: 'Posts with High Comment Count by User' }) .run();
Click "Complete Pipeline" to see the result
